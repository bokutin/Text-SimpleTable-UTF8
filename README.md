# NAME

Text::SimpleTable::UTF8 - Text::SimpleTable のUTF8対応版です。

# SYNOPSIS

.-------------------------------------+------------.
| Parameter                           | Value      |
\+-------------------------------------+------------+
| hello                               | goodbye    |
| こんにちは                          | さようなら |
'-------------------------------------+------------'

.-------------------------------------+-----.
| Parameter                           | Va- |
|                                     | lue |
\+-------------------------------------+-----+
| hello                               | go- |
|                                     | od- |
|                                     | bye |
| こんにちは                          | さ- |
|                                     | よ- |
|                                     | う- |
|                                     | な- |
|                                     | ら  |
'-------------------------------------+-----'

.-------------------------------------+------.
| Parameter                           | Val- |
|                                     | ue   |
\+-------------------------------------+------+
| hello                               | goo- |
|                                     | dbye |
| こんにちは                          | さy- |
|                                     | oう- |
|                                     | なら |
'-------------------------------------+------'
